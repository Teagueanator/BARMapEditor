# Keyboard shortcuts

This is the canonical reference. The Sprint 22 cheat-sheet (`?`)
still works and surfaces the same content as a compact modal.

## Tools

| Key | Tool |
|-----|------|
| Q   | Select / orbit |
| B   | Sculpt |
| S   | Start positions |
| M   | Metal spots |
| V   | Geo vents |
| F   | Features |
| W   | Water / Lava |
| L   | Paint layer |
| G   | Procgen |

## Camera + canvas

| Combo | Action |
|-------|--------|
| LMB drag (Sculpt) | Apply brush |
| LMB drag (others) | Tool-specific gesture (see tool article) |
| RMB drag | Orbit camera |
| MMB drag | Pan |
| Scroll | Zoom |
| Arrow keys | Pan (Shift = 5× faster) |

## Project / help

| Combo | Action |
|-------|--------|
| Ctrl+Z | Undo |
| Ctrl+Shift+Z / Ctrl+Y | Redo |
| Ctrl+S | Save |
| Ctrl+Shift+S | Save as… |
| Ctrl+B | Build & install (reserved) |
| Ctrl+K | Command palette |
| Ctrl+Shift+H | What's-this hover-popover mode |
| ? | Cheat sheet |
| F9 | Mapinfo form |
| Esc | Close modal / cancel mode |

The command palette (`Ctrl+K`) makes every shortcut and menu item
fuzzy-searchable. If you can name an action, you can find it
there.

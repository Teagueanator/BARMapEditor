# Select / orbit

The default tool — camera only, no canvas editing. Use it when
you want to inspect terrain, plan placements, or just navigate
without risking an accidental stamp.

LMB orbits, MMB pans, RMB orbits, and the scroll wheel zooms.
Arrow keys pan the camera with Shift modifier for 5× speed.

To start editing, pick a tool from the left strip or use one of
the keyboard accelerators: `B` Sculpt, `S` Start positions, `M`
Metal spots, `V` Geo vents, `F` Features, `W` Water, `L` Paint
layer, `G` Procgen.

## Why a separate Select tool?

Sculpt-mode LMB stamps the brush, which is dangerous if you just
want to look around. Select gives you an explicit "look, don't
touch" mode. The Inspector under Select shows the global project
context (size, symmetry, validation chip) so you can sanity-check
the map before going back to editing.
